Правила использования

Как применять палитру правильно.

✓ Основной текст: slate-700 (#334155) на светлом фоне, slate-200 (#e2e8f0) на тёмном
✓ Фоны: slate-50 (#f8fafc) для основы, slate-100 (#f1f5f9) для карточек, белый (#ffffff) для модалок
✓ Акценты: slate-900 (#0f172a) для кнопок и важных элементов
✓ Семантические цвета: используй только для статусов — teal (успех), rose (ошибка), orange (предупреждение)
⚠ Не перегружай цветом: максимум 1-2 акцентных элемента на экране
✗ Не используй: яркие цвета для фонов, слишком контрастные сочетания, градиенты без необходимости
